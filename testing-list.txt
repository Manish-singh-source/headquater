
1. Dashboard: 

=================================== 
1. Access Control: 
    - Only authorized personnel should have access to the view, edit, delete, and download functionalities.
    
    Staffs: 
    - View: view All staff members 
        - view in detail staff member 
        - view staff member's related permissions
    - Edit: edit staff member details 
    - Delete: delete staff member
    - Update Status: update status of staff member 
    - multi delete: delete multiple staff members at once
    - multi update status: update status of multiple staff members at once 
    - add new staff member: add a new staff member to the system 
    - Download: download staff member details in excel format 

    Add Staff Rules: 
    - role and warehouse are required fields
    - email must be unique and in a valid email format
    - password must meet security requirements (e.g., minimum length, complexity) 
    - phone number must be in a valid format

    - Edit: edit warehouse details

    - multi update status: update status of multiple warehouses at once only status of non-default warehouses can be updated

1. Sales Orders: 
    - Check Availability: check product availability in stock before creating a sales order
        - Rules: 
            - only show active warehouses in dropdown 
            - only show active customer groups in dropdown
            - Upload excel file with specified format containing: 
                - columns list: Product SKU, Quantity
            - First: check all columns are present 
            - Second: check customer (facility name same as customer facility name) exists in file & system  
            - Third: check product SKUs exist in file & system 
            - Then proceed to check stock availability for each product SKU
            - check required quantity is available in stock for each product SKU 
            - generate report with results: 
                - excel file with columns: Product SKU, Requested Quantity, Available Quantity, Status (Available/Not Available) 

    Note: 
        - For creating Sales Order: add block quantity in excel file format to hold quantity for creating sales order

    - Create Sales Order: create a new sales order in the system
        - Rules: 
            - every thing as check availability
            - check if vendor exists in system 
            - generate sales order 
            - generate purchase order if any product is not available in stock 

    - Sales Order List: 
        - list of All sales orders 
            - tab switching for: 
                - All Sales Orders
                - Blocked Sales Orders
                - Completed Sales Orders 
                - Ready to Package Sales Orders
                - Ready to Ship Sales Orders 
                - Shipped Sales Orders
                - Delivered Sales Orders
                - Cancelled Sales Orders
        - delete sales order
        - view sales order details        
                - tab switching: 
                    - all products in sales order
                    - packaging products in sales order 
                    - packaged products in sales order 
                    - shipped products in sales order
                    - delivered products in sales order
                    - cancelled products in sales order 

                - sales order products list 
                - delete selected sales order product
                - download sales order products in excel format
                - update sales order products using downloaded excel format 
            - update sales order status 
            - generate invoice for sales order
