# Manual Invoice

The **Manual Invoice** section is used to create an invoice manually when the invoice is not generated from a sales order.

---

## Create Manual Invoice

To create a manual invoice:


![Manual Invoice](/docsimages/mi.png)


1. Open the main menu.
2. Click **Invoice**.
3. Open the **Manual Invoice** tab.
4. Click **Create Manual Invoice**.

![Manual Invoice](/docsimages/mi2.png)
5. The system will show the invoice form.
6. Select the required **Customer**.
7. Select the **Invoice Type**:
   - Select **Product** if the invoice is for products.
   - Select **Service** if the invoice is for a service.
   - Select **Marketing** if the invoice is for marketing.
8. Select the **Payment Mode**.
9. Enter the invoice **Amount**.
10. Click **Create Invoice**.

![Manual Invoice](/docsimages/mi4.png)

After submitting the form, the system will create the manual invoice.

---

## View Manual Invoice

After the manual invoice is created:

1. Open the **Invoice** section from the menu.
2. Click the **Manual Invoice** tab.
3. The created invoice will appear in the invoice list.

![Manual Invoice](/docsimages/mi5.png)

---

## Summary

The manual invoice process follows these steps:

1. Open **Invoice** from the menu.
2. Go to the **Manual Invoice** tab.
3. Click **Create Manual Invoice**.
4. Select the customer.
5. Select the invoice type.
6. Select the payment mode.
7. Enter the amount.
8. Create the invoice.
9. View the created invoice in the **Manual Invoice** list.
