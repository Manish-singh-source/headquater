# Ready to Ship

The **Ready to Ship** tab is used to view and process orders that are approved and ready for shipment. After packaging is completed and approved, the order appears in this section so the admin can review the shipping details and mark the order as **Shipped**.

---

## Purpose of Ready to Ship

The Ready to Ship process helps the admin:

- View all orders that are ready for shipment.
- Check shipment-ready orders in a table list.
- Open the required sales order before shipping.
- Review order details based on the batch ID.
- Confirm the shipping order and shipment details.
- Change the order status from **Ready to Ship** to **Shipped**.

---

## Open Ready to Ship

![Ready to Ship List](/docsimages/Ready-ship-list.png)

To open the Ready to Ship section:

1. Open the main menu.
2. Click **Ready to Ship**.
3. The system will show a table list of orders that are ready for shipment.

Use this list to identify which order needs to be shipped.

---

## Select the Order

After opening the Ready to Ship list:

1. Find the sales order that needs to be prepared for shipment.
2. Review the basic order information shown in the table.
3. Open the required order.

After opening the order, the system will show the order details based on the related batch ID.

---

## Understand Batch ID Wise Shipment

![Shipped Batch](/docsimages/shipped-batch.png)

Orders are managed batch ID wise in the Ready to Ship section.

For example:

- A sales order may have products assigned to **Warehouse 1**.
- Packaging may be started for that warehouse.
- The same sales order may also have packaging started in another warehouse.
- Because each warehouse packaging process is separate, the shipment is also handled separately by batch ID.

This means one sales order can have different packaging and shipment records if the products are handled by different warehouses or different packaging batches.

The batch ID helps the admin identify which packaged batch is ready to be shipped.

---

## View Shipping Details

After opening the required order:

1. Review the order information.
2. Check the shipping order details.
3. Check the customer and destination details.
4. Confirm that the correct batch ID is being shipped.
5. Verify that the order is ready before changing the status.

Only ship the order after confirming the shipping and destination details.

---

## Change Status to Shipped

![Shipped Status](/docsimages/shipped-status.png)

To mark the order as shipped:

1. Open the required order from the **Ready to Ship** list.
2. Review the order, batch ID, shipping, and destination details.
3. Click the **Change Status** button at the top of the page.
4. Select **Shipped**.
5. Save or confirm the status change.

After this step, the order status will be updated to **Shipped**.

---

## Summary

The Ready to Ship process follows these steps:

1. Open **Ready to Ship** from the main menu.
2. View the table list of shipment-ready orders.
3. Identify which order needs to be shipped.
4. Open the required order.
5. Review the order details based on the batch ID.
6. Check shipping order details and destination details.
7. Click **Change Status** at the top of the page.
8. Change the status to **Shipped**.
9. The order is now marked as shipped.
