# Vendors

The **Vendors** page is used to create and manage vendor records. Vendor details are used in purchase orders, vendor PI, GRN, and purchase reports.

---

## Open Vendors

![Vendors List](/docsimages/v1.png)

To open the Vendors page:

1. Open the main menu.
2. Go to **Master**.
3. Click **Vendors**.
4. The system will show the **Vendors** page.

The Vendors page is available from the `/vendors` URL.

---

## Vendors List

The vendor list shows all vendor records with contact, order, location, and status details.

The table includes:

1. **SR.No**
2. **Vendor Code**
3. **Vendor Name**
4. **Email**
5. **Contact Number**
6. **Orders**
7. **Location**
8. **Joined At**
9. **Status**
10. **Action**

---

## Add Vendor

![Vendors List](/docsimages/v2.png)

To add a vendor:

1. Click **Add Vendor**.
2. Enter the vendor details:
   - Vendor code
   - Client name
   - Contact name
   - Phone
   - Email
   - PAN number
   - GSTIN
   - GST treatment
3. Enter the **Shipping Address** details.
4. Enter the **Billing Address** details.
5. Select the vendor **Status**.
6. Save the vendor.

---

## Manage Vendor

![Vendors List](/docsimages/v3.png)

Use the action buttons to manage vendors:

1. Click **View** to open vendor details.
2. Click **Edit** to update vendor information.
3. Click **Delete** to remove a vendor.
4. Use the status option to activate or deactivate a vendor.
5. Use **Change Status** from the action menu to update selected vendors together.

---

## Summary

The Vendors process follows these steps:

1. Open **Master > Vendors**.
2. Review the vendor list.
3. Click **Add Vendor** to create a vendor.
4. Fill vendor, shipping, and billing details.
5. View, edit, delete, or update status from the action buttons.
