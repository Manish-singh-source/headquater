# Warehouses

The **Warehouses** page is used to create and manage warehouse records. Warehouse details are used for stock, dispatch, receiving, return, and inventory operations.

---

## Open Warehouses

![Warehouses List](/docsimages/w1.png)

To open the Warehouses page:

1. Open the main menu.
2. Go to **Master**.
3. Click **Warehouses**.
4. The system will show the **Warehouse** page.

The Warehouses page is available from the `/warehouses` URL.

---

## Warehouses List

The warehouse list shows all warehouse records.

The table includes:

1. **Warehouse Id**
2. **Warehouse Name**
3. **Type**
4. **City**
5. **Contact Person Name**
6. **Phone Number**
7. **Default Warehouse**
8. **Status**
9. **Action**

---

## Add Warehouse

![Warehouses List](/docsimages/w2.png)

To add a warehouse:

1. Click **Add Warehouse**.
2. Enter the warehouse details:
   - Warehouse name
   - Warehouse type
   - Contact person name
   - Phone number
   - Alternate phone number
   - Email
   - GST number
   - PAN number
3. Enter address details.
4. Upload the licence document if required.
5. Enter the maximum storage capacity.
6. Select country, state, city, and pin code.
7. Select warehouse status.
8. Select supported operations.
9. Mark **Default Warehouse** if this warehouse should be the default.
10. Save the warehouse.

---

## Manage Warehouse

![Warehouses List](/docsimages/w3.png)

Use the action buttons to manage warehouses:

1. Click **View** to open warehouse details.
2. Click **Edit** to update warehouse information.
3. Click **Delete** to remove a warehouse.
4. Use the status option to activate or deactivate a warehouse.
5. Use the bulk action menu to update or delete selected warehouses.

---

## Summary

The Warehouses process follows these steps:

1. Open **Master > Warehouses**.
2. Review the warehouse list.
3. Click **Add Warehouse** to create a warehouse.
4. Fill contact, address, capacity, and operation details.
5. View, edit, delete, or update status from the action buttons.
