# Dashboard

The **Dashboard** page shows the main analytics and status summary for the business. It helps the admin review sales, purchases, orders, dispatch, delivery, GRN, payment, and warehouse inventory from one place.

---

## Open Dashboard

![Dashboard](/docsimages/d1.png)

To open the dashboard:

1. Open the main menu.
2. Click **Dashboard**.
3. The system will show the analytics dashboard.

The dashboard is available for users who have permission to view dashboard analytics.

---

## Apply Dashboard Filters

![Dashboard Filters](/docsimages/d2.png)

The dashboard provides filters to review data for a selected period and brand.

To apply filters:

1. Select the **Start Date**.
2. Select the **End Date**.
3. Select a brand from **Select Brands**.
4. Click **Apply**.

To clear the selected filters, click **Reset**.

---

## Sales Analytics

![Sales Analytics](/docsimages/d3.png)

The **Sales Analytics** section shows sales performance for the selected filter range.

This section includes:

1. **Total Sales Till Date** for the current year.
2. **Sales Trend** for the last four months.
3. **Sales by Brand** with brand-wise sales amount.

Use this section to compare sales performance by brand and month.

---

## Purchase Analytics

![Purchase Analytics](/docsimages/d4.png)

The **Purchase Analytics** section shows purchase performance for received vendor PI products.

This section includes:

1. **Total Purchases Till Date** for the current year.
2. **Purchase Trend** for the last four months.
3. **Purchases by Brand** with brand-wise purchase amount.

Use this section to review how much stock has been purchased by brand.

---

## Order Status

![Order Status](/docsimages/d5.png)

The **Order Status** section shows the current status of sales orders.

This section includes:

1. **Total Orders**.
2. **Open Orders**.
3. **Processed Orders**.
4. **Orders by Brand**.
5. **Processed Percentage** for each brand.

Use this section to check pending and completed order performance.

---

## Dispatch Status

![Dispatch Status](/docsimages/d6.png)

The **Dispatch Status** section shows dispatch-related pending activities.

This section includes:

1. **LR Pending**.
2. **Appointment Received & GRN Pending**.
3. **Appointment Pending**.

Some count values are clickable and can open the related report page.

---

## Delivery Confirmation

The **Delivery Confirmation** section shows POD status.

This section includes:

1. **POD Received**.
2. **POD Not Received**.

Use this section to track delivery confirmation completion.

---

## GRN Status

![GRN Status](/docsimages/d7.png)

The **GRN Status** section shows whether GRN details are completed for delivered orders.

This section includes:

1. **GRN Done**.
2. **GRN Not Done**.

Use this section to identify invoices or orders where GRN details are still pending.

---

## Payment Status

The **Payment Status** section shows invoice payment information.

This section includes:

1. **Total Outstanding**.
2. **Monthly Received**.
3. **Due**.
4. **Payment Trend** for the last four months.

Use this section to review pending payments and received payment amounts.

---

## Warehouse Inventory

![Warehouse Inventory](/docsimages/d8.png)

The **Warehouse Inventory** section shows stock availability and inventory value.

This section includes:

1. **Total Inventory Units**.
2. **Total Inventory Value**.
3. **Inventory by Brand**.
4. Brand-wise inventory units and inventory value.

The inventory unit count can be used to open the related product list.

---

## Summary

The dashboard process follows these steps:

1. Open **Dashboard** from the menu.
2. Select the required date range.
3. Select a brand if required.
4. Click **Apply**.
5. Review sales and purchase analytics.
6. Check order, dispatch, delivery, and GRN status.
7. Review payment outstanding and monthly received amounts.
8. Check warehouse inventory units and value.
