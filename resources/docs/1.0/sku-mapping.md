# SKU Mapping

The **SKU Mapping** page is used to manage mappings between SKU codes, portal codes, item codes, and pricing details. This helps match customer or portal item codes with internal product SKUs.

---

## Open SKU Mapping

![SKU Mapping List](/docsimages/sk1.png)

To open the SKU Mapping page:

1. Open the main menu.
2. Go to **Master**.
3. Click **SKU Mapping**.
4. The system will show the **SKU Mapping** page.

The SKU Mapping page is available from the `/sku-mapping` URL.

---

## SKU Mapping List

The SKU Mapping list shows all mapping records.

The table includes:

1. **SKU**
2. **Portal Code**
3. **Item Code**
4. **Basic Rate**
5. **Net Landing Rate**
6. **MRP**
7. **Action**

---

## Download Excel

![SKU Mapping List](/docsimages/sk2.png)

To download the SKU mapping Excel file:

1. Click **Download Excel**.
2. The system will download the SKU mapping Excel format or existing mapping data.
3. Open the file and review the headings.

---

## Upload SKU Mapping Excel

![SKU Mapping List](/docsimages/sk3.png)

To upload SKU mapping data:

1. Click **Upload Excel**.
2. Select the SKU mapping file in CSV or XLSX format.
3. Submit the upload form.

Use **Add SKU Mapping (Excel)** when new mapping records need to be added in bulk.

---

## Manage SKU Mapping

![SKU Mapping List](/docsimages/sk4.png)

Use the action buttons to manage SKU mappings:

1. Click **Edit** to update a mapping record.
2. Click **Delete** to remove a mapping record.
3. Use Excel upload when multiple mappings need to be added together.

---

## Summary

The SKU Mapping process follows these steps:

1. Open **Master > SKU Mapping**.
2. Review existing mappings.
3. Download the Excel format if required.
4. Upload or add mappings through Excel.
5. Edit or delete individual mapping records if required.
