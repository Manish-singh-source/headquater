# Excel Formats

The **Excel Formats** section is used to download predefined Excel templates for bulk data entry and upload.

---

## Open Excel Formats

To open the Excel Formats section:

   ![Downloaded Invoice](/docsimages/ms.png)

1. Open the main menu.
2. Click **Excel Formats**.
3. The system will show the available Excel format templates.

---

## Download Excel Format

   ![Downloaded Invoice](/docsimages/ms1.png)

In this section, Excel formats are available for different categories.

To use an Excel format:

1. Find the required category from the list.
2. Download the Excel format.
3. Open the downloaded Excel file.
4. The Excel file will already contain the required headings.
5. Add the data under the correct headings.
6. Save the file.
7. Use the completed file for the related bulk upload or import process.

---

## Available Excel Formats

The following Excel formats are available:

1. **Customers Group**
2. **Customers Bulk Upload**
3. **Products**
4. **SKU Mapping**
5. **Custom Customer PO**
6. **Customer PO**
7. **Customer Return**
8. **Customer PO Block Quantity**
9. **Vendor PO**
10. **Vendor PI**

---

## Summary

The Excel Formats process follows these steps:

1. Open **Excel Formats** from the menu.
2. Select the required category.
3. Download the Excel format.
4. Fill the data under the provided headings.
5. Save the Excel file.
6. Upload or use the file in the related module.
